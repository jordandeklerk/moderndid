Extended TWFE (Wooldridge)
==========================

The ETWFE module implements the Extended Two-Way Fixed Effects methodology
for staggered difference-in-differences with heterogeneous treatment effects,
based on `Wooldridge (2023) <https://doi.org/10.1093/ectj/utad016>`_ and
`Wooldridge (2025) <https://doi.org/10.1007/s00181-025-02807-z>`_.
